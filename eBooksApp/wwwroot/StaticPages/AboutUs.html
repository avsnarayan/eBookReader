About Us

This is an eBook Reader App.

It consists of following features:
  • Website with two sections: a public static section using HTML5, CSS3, and ES6, and a password protected section using Asp.Net Core MVC.
  • Ability to select and view both publicly available documents, and, with authentication, private documents.
  • Controls to change the viewing scale and to iterate through pages or sections of any of the documents.
  • Ability to comment on documents for review, associated with a document being reviewed.
  • Webservice built using Asp.Net Web API, used to manage content on the site.
  • Clients for WebService.
  • Ability to upload, replace, and delete any of the stored documents. Sources for the documents may be any file of a supported format from the local file system.
